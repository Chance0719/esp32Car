# esp32Car

#### 介绍
基于esp32的两轮平衡小车

#### 软件架构
c++，arduino

#### 硬件模块
esp32开发版*1、亚克力板*2、mpu6050陀螺仪*1、0.96寸ssd1306显示屏（i2c协议）*1、dc-dc可调降压模块*2、DRV8838直流电机驱动版*1、65mm
橡胶轮胎*2、18650电池*3、3位串联电池盒*1、JGA25-370 6v直流减速电机*2、电机固定支架*2、轮胎连接轴*2、开关*1、亚克力连接柱*4、螺丝螺母若干、杜邦线若干
